# glw
<https://github.com/vreui/glw>

> glw = glutin + winit

[![CI](https://github.com/vreui/glw/actions/workflows/ci.yml/badge.svg)](https://github.com/vreui/glw/actions)

Less feature, less BUG.


## 编译开发

+ 代码格式化:

  ```
  cargo fmt
  ```

+ 编译:

  ```
  cargo build
  ```

+ 测试:

  ```
  cargo test
  ```


## 平台支持

glw (计划) 支持下列平台 (协议):

+ `wayland` (GNU/Linux)

+ `android` 9+

+ `windows` 7+

+ `web` (wasm)
